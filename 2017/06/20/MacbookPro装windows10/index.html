MacbookPro装windows10

发表于 2017-06-20 | 分类于 软实力 | 热度 ℃ |

简介

因为想体现下dota2的精美画面，自己大学时候的windows机器的显卡（集成显卡）太差。就在我的MacBookPro下装了windows。之前大学时候可能会觉得买mac然后装windows系统感觉很low。但是现在的自己竟然成为了当初自己认为很low的那个人。想想还是有点有趣的。

Boot Camp助理

Boot Camp助理是macpro 自带的一个装双系统的软件，真是不得不佩服mac的开发者是这么贴心，连装windows的用户都提供了软件。我们要做的只是下载好win10的镜像文件：

第1步

在应用程序/实用工具文件夹中打开Boot Camp助理（或使用右上角的搜索功能找到它），然后点击“ 继续 ”。

第2步

使用“ 选择… ”按钮找到您在第1部分下载的Windows .ISO文件（默认情况下在您的下载文件夹里）。

第3步

使用滑块来调整硬盘空间的大小或者点击“ 均分 ”，以满足您的要求，来把硬盘分给win10。，当然，也可以通过分区拖动滑块到左边给Windows的更多的空间。然后点击“ 安装 ”

第4步

Boot Camp助理要一些时间来复制Windows文件并下载Windows支持软件。 （根据您的宽带速度，这可能需要一段时间。）

最后

bootcamp助理工作完成后，会自动重启进入安装界面。
也可以一直按住option键 然后现在windows。
接下去我就默认大家都会装window10的过程

碰到的问题

辛酸泪，一些坑吧。

缺少驱动

windows10安装完成后，无法连接上网络，缺少windows下的驱动。

  • 打开windows文件资源管理器
  • 进入目录里的 BootCamp 目录下，双击 Setup 可以为系统安装苹果驱动。

硬盘分配过小

  • 进入mac系统
  • 打开磁盘管理器
  • 点击分区，选择合适的新建分区大小
  • 切换进入windows10 打开磁盘管理器，然后找到新增的磁盘，格式化称ntfs格式，然后压缩卷成为新的windows10磁盘

开机按Option键。

这种方法是使用最多的方法，也是最常用的方法。这个其实很简单，就是开机之后，一直按着Option键，知道出现系统选择界面，按左右方向键选择我们需要启动的系统，选中之后，直接鼠标左键点击，或者回车键确认即可。
