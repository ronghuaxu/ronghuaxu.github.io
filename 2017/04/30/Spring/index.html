简介

让我们建立一个更好的企业。
Spring帮助各地的开发团队构建简单，便携，快速和灵活的基于JVM的系统和应用程序。

特点

建立任何类

为您选择的基础架构组件编写干净，可测试的代码，并完成任务，而无需重新创建轮子。

兼容任何平台

保持便携式 - 基于Spring的应用程序可以在JVM所在的任何地方运行。 在云端部署或上述所有功能部署独立的应用服务器。

用的放心

代码有信心 - Spring提供了一个开放的编程模型，是全面，凝聚力，广泛的理解和良好的支持。

Spring Tool Suite™

Spring Tool Suite是一个基于Eclipse的开发环境，为开发Spring应用程序而定制。 它提供了一个即用的环境来实现，调试，运行和部署您的Spring应用程序，包括Pivotal tc服务器，Pivotal Cloud Foundry，Git，Maven，AspectJ的集成，并在最新的Eclipse版本之上。

包含在Spring Tool Suite中的是Pivotal tc Server的开发人员版本，它是为Spring优化的Apache Tomcat的替代品。 通过其Spring Insight控制台，tc Server Developer Edition提供了应用程序性能指标的图形实时视图，使开发人员能够从桌面识别和诊断问题。

Spring Tool套件支持将应用程序定位到本地，虚拟和基于云的服务器。 它可以免费提供开发和内部业务操作，没有时间限制，完全开放源代码，并根据Eclipse公共许可证的条款许可。
