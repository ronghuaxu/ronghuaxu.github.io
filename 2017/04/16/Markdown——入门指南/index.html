易读易写!
Markdown 是一个 Web 上使用的文本到HTML的转换工具，可以通过简单、易读易写的文本格式生成结构化的HTML文档。目前 github、Stackoverflow 等网站均支持这种格式。

Mac和Windows下的编辑软件推荐

Mac 平台 ：MacDown is open source Markdown editor for macOS
Windows 平台 ： MarkdownPad is a full-featured Markdown editor for Windows.
